### 批次 101 — role.html / permission.html 入口 admin seed

**文件**: `role.html`、`permission.html`

- `init()` 中串联：
  ```js
  RT_USERS.ensureDefaultAdminRole({ account: 'admin', password: '123', nickname: '管理员', operator: 'system' })
    .then(() => RT_PERMISSIONS.seedMenusFromRegistry('system'))
  ```

作用：
- 角色管理页入口 → 自动创建「系统管理员」角色 + 绑定 admin 账号
- 权限管理页入口 → 保障菜单注册表播种

### 批次 102 — 编码规范：UTF-8 无 BOM（强制）

**文件**: `RULES.md`、`.githooks/pre-commit`

- RULES.md 新增 §6 文件编码规则：
  - 所有源文件（`.html`/`.js`/`.css`/`.md`/`.json`）必须 UTF-8 无 BOM
  - 禁止 `U+FFFD` 替换字符
- 新增 `pre-commit` hook：扫描暂存文件，检测到 `\xef\xbf\xbd` 拒绝提交
- 安装方式：`ln -sf ../../.githooks/pre-commit .git/hooks/pre-commit`
